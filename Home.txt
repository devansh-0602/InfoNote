            <h3 id="t1"> head : </h3>
            <p style="color: black">
            The head element is a container for metadata (data about data) and is placed between the html tag and the body tag.<br>
            Metadata is data about the HTML document. Metadata is not displayed.<br>
            Metadata typically define the document title, character set, styles, scripts, and other meta information.
            </p>
            <h4><i>Example</i> : </h4>
            <xmp>
            <html>
            <head>
            <title>Title of the document</title>
            </head>
            </html>
            </xmp>
        </div>
        <div class="course-col">
            <h3 id="t2"> body : </h3>
            <p style="color: black">
            The <body> tag defines the document's body.<br>
            The <body> element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.
            </p>
            <h4><i>Example</i> : </h4>
            <xmp>
            <html>
            <head>
            <title>Title of the document</title>
            </head>
            <body>
            <h1>This is a heading</h1>
            <p>This is a paragraph.</p>
            </body>
            </html>
            </xmp>

            <p style="color: black">
            The p tag defines a paragraph.<br>
            Browsers automatically add a single blank line before and after each p element.
            </p>
            <h4><i>Example</i> : </h4>
            <xmp>
            <html>
            <body>
            <h1>The p element</h1>

            <h3 id="t4"> ol : </h3>
            <p style="color: black">
            The ol tag defines an ordered list. An ordered list can be numerical or alphabetical.<br>
            The li tag is used to define each list item.
            </p>
            <h4><i>Example</i> : </h4>
            <xmp>
            <html>
            <body>
            <h1>The ol element</h1>
            <ol>
            <li>Coffee</li>
            <li>Tea</li>
            <li>Milk</li>
            </ol>
            <ol start="50">
            <li>Coffee</li>
            <li>Tea</li>
            <li>Milk</li>
            </ol>
            </body>
            </html>
            </xmp>

            <h1 id="t5"> li : </h1>
            <p style="color: black">
            The li tag defines a list item.
            The li tag is used inside ordered lists(ol), unordered lists (ul), and in menu lists (menu).
            <br> In ul and menu, the list items will usually be displayed with bullet points.
            In ol, the list items will usually be displayed with numbers or letters.
            </p>
            <h2><i>Example</i> : </h2>
            <xmp>
            <html>
            <body>
            <h1>The ol and ul elements</h1>
            <p>The ol element defines an ordered list:</p>
            <ol>
            <li>Coffee</li>
            <li>Tea</li>
            <li>Milk</li>
            </ol>
            </body>
            </html>

            <p style="color: black">

            <xmp>
            <html>

            <h1 id="t12"> del : </h1>
            <p style="color: black">
            The del tag defines text that has been deleted from a document. <br>Browsers will usually strike a line through deleted text.

            <xmp>
            <html>

            <h1 id="t14"> sub(subscript) AND sup(superscript) : </h1>
            <p style="color: black">
            The HTML sub element defines subscript text. <br>
            Subscript text appears half a character below the normal line, and is sometimes rendered in a smaller font. <br>
            <b>AND</b>
            <br>The HTML sup element defines superscript text. 
            </br>Superscript text appears half a character above the normal line, and is sometimes rendered in a smaller font. 
            </p>
            <h2><i>Example</i> : </h2>
            <xmp>
            <html>
            <body>
            <p>
            The chemical formula of water is H<sub>2</sub>O.
            <br><b>AND</b><br>
            4<sup>2</sup> = 16.
            </p>
            </body>
            </html>

            <p style="color: black">
            The HTML style attribute is used to add styles to an element, such as color, font, size, and more.

            <h1 id="t16">table : </h1>
            <p style="color: black">
            The table tag defines an HTML table.<br>An HTML table consists of one table element and one or more tr, th, and td elements.<br>
            The tr element defines a table row, the th element defines a table header, and the td element defines a table cell.
            </p>
            <h2><i>Example</i> : </h2>
            <xmp>
            <html>
            <head>
            <title>HTML table Tag</title>
